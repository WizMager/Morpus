# Morpus

## Геймплей

Игра в жанре survival/roguelike, где игрок управляет персонажем и автоматически атакует ближайших врагов в радиусе действия.

Есть кнопка, нажимая на котрую можно улучшит характеристики:
- Скорость передвижения
- Урон в секунду
- Радиус атаки

Враги постоянно спавнятся вокруг игрока, создавая динамичный геймплей.

## Управление

- **WASD** или **Стрелки** - движение персонажа

## Технологический стек

- **Unity** (Universal Render Pipeline)
- **Morpeh** (ECS фреймворк)
- **Unity Input System**
- **UniRx**
- **Cinemachine**
- **TextMeshPro**
